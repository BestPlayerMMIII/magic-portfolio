## State Management

### Admin Panel (TipTap Extension)

```javascript
// Internal state stored in node attributes
{
  id: "BUTTON",                  // Toolcall identifier
  "data-params": "{               // JSON-encoded parameters
    text: 'Get Started',
    href: '/signup',
    style: 'primary'
  }"
}

// Rendered HTML output
<gitcms-toolcall
  id="BUTTON"
  _text="Get Started"
  _href="/signup"
  _style="primary">
</gitcms-toolcall>
```

### Client SDK (Resolution)

```typescript
// Input: HTML with toolcalls
const content = `
  <p>Welcome!</p>
  <gitcms-toolcall id="BUTTON" _text="Get Started" _href="/signup"></gitcms-toolcall>
`;

// Extraction: Parse to structured data
const toolcalls = extractToolcalls(content);
// [{
//   id: "BUTTON",
//   parameters: {
//     text: "Get Started",
//     href: "/signup"
//   },
//   originalTag: "<gitcms-toolcall...>"
// }]

// Resolution: Apply renderers
const renderers = {
  BUTTON: (id, params) => `<a href="${params.href}">${params.text}</a>`,
};

// Output: Processed HTML
const result = resolveToolcalls(content, renderers);
// "<p>Welcome!</p><a href="/signup">Get Started</a>"
```

### Input Sanitization

```typescript
// âŒ Bad: Direct interpolation

// âœ… Good: Sanitize HTML
```

### Parameter Validation

```typescript
// âœ… Validate required parameters
BUTTON: (id, params) => {
  if (!params.text || !params.href) {
    return '<span class="error">Invalid button</span>';
  }
  return `<a href="${params.href}">${params.text}</a>`;
};
```

### URL Validation
